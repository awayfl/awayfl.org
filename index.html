AwayFL

AwayFL is an open source Adobe Flash Player emulator, allowing the playback of .SWF files natively in a browser. It is written in Typescript and is compatible with ES3, ES5 and ES6 Javascript outputs.

  • TEST YOUR SWF

Features

Supported

ActionsScript 1, 2 & 3 execution 90%
Timeline playback & scripting 100%
MovieClip, Button, Graphic and Textfield primitives 100%
Scripted graphics API 90%
Dynamic masks (timeline & script based) 100%
Sound manipulation (AS2 & AS3) 80%

In-Development

Flash 8 filter effects 50%
Photoshop-style blend modes 50%
Adobe Text Layout Framework (TLF) 40%
Stage3D APIs & Texture formats 10%
H.264 Video streaming 5%
Camera & microphone input 5%
